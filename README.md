# Introdução

O **Shortener** é uma API projetada para encurtar URLs longas, inspirada no funcionamento do site [encurtador.dev](https://www.encurtador.dev/). Este projeto tem como objetivo replicar e aprimorar o que imagino ser o backend do [encurtador.dev](https://www.encurtador.dev/), oferecendo uma solução self-hosted para usuários que desejam controle total sobre o serviço.

Desenvolvido inteiramente em Golang, o Shortener utiliza bancos de dados SQL para armazenamento persistente e um banco de dados NoSQL para cache, garantindo eficiência, escalabilidade e desempenho.
